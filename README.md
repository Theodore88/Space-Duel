# Space-Duel
A competitive 2-D two player arcade game created with python.  

## Controls
### Black Ship
A - Move Left\
D - Move Right\
W - Move Up\
S - Move Down\
Spacebar - Shoot

### White Ship
Left Arrow Key - Move Left\
Right Arrow Key - Move Right\
Up Arrow Key - Move Up\
Down Arrow Key - Move Down\
Right CTRL - Shoot
